## 🎯 Sobre o Projeto

O **Agente Concurseiro** é uma plataforma completa e moderna para preparação de concursos públicos, que utiliza **Inteligência Artificial** e **gamificação** para maximizar o aprendizado e a motivação dos candidatos. O sistema foi projetado para ser robusto, escalável e pronto para produção.

### 🌟 Diferenciais

-   **Especialização por Banca**: Avaliação e conteúdo direcionado para as principais bancas (CESPE, FCC, VUNESP, FGV, IBFC).
-   **IA Avançada**: Integração com GPT-4 para análises, predições e recomendações personalizadas.
-   **Gamificação Científica**: Sistema motivacional completo com conquistas, badges, níveis e ranking.
-   **Analytics Profissionais**: Predição de desempenho com mais de 85% de precisão e painéis interativos.
-   **Infraestrutura Enterprise**: 100% containerizado com Docker, pronto para alta disponibilidade.

---

## 🚀 Funcionalidades Principais

O sistema conta com 23 funcionalidades principais implementadas, das quais se destacam:

-   **📝 Sistema de Redação Avançado**: Avaliação específica por banca com feedback detalhado.
-   **🎯 Simulados Adaptativos**: Questões reais com dificuldade ajustável e análise de desempenho.
-   **📊 Analytics e Predição com IA**: Painel com 8 métricas de performance, predição de nota final e simulação de cenários.
-   **🎮 Sistema de Gamificação Completo**: 15 conquistas, 9 badges, sistema de XP, sequências de estudo e ranking.
-   **📚 Planos de Estudo Personalizados**: Cronogramas gerados por IA baseados no perfil e objetivos do candidato.
-   **🔔 Sistema de Notificações Inteligente**: 7 tipos de notificações contextuais para manter o engajamento.

## 🛠️ Instalação e Execução

### Pré-requisitos
-   Python 3.11+
-   Git
-   Docker (Opcional, para produção)

### Instalação Rápida
```bash
# 1. Clone o repositório
git clone https://github.com/clenio77/agente_concurseiro.git
cd agente_concurseiro

# 2. Execute o script de instalação simplificada
chmod +x install-simple.sh
./install-simple.sh

# 3. Inicie a aplicação
python3 run_app.py
```

Após a execução, a interface estará disponível em `http://localhost:8501`.

## 🕹️ Como Usar

1.  **Acesse a plataforma**: Abra seu navegador e acesse `http://localhost:8501`.
2.  **Crie sua conta**: Faça o registro para ter uma experiência personalizada.
3.  **Configure seu Perfil**: Defina seu concurso, banca e metas de estudo.
4.  **Explore as Funcionalidades**:
    -   Gere seu **plano de estudos** personalizado.
    -   Faça **simulados** para testar seus conhecimentos.
    -   Envie **redações** para avaliação.
    -   Acompanhe seu progresso no **dashboard de analytics**.
    -   Mantenha-se motivado com o **sistema de gamificação**.
